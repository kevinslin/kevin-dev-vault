### Generate public key from private

- source: [^1]
```bash
$ ssh-keygen -y -f ~/.ssh/id_rsa > ~/.ssh/id_rsa.pub
```

[^1]: https://blog.tinned-software.net/generate-public-ssh-key-from-private-ssh-key/
